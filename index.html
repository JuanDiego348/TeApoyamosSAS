Análisis del Problema y Necesidades Clave

La firma de abogados "Te Apoyamos SAS" necesita expandir su negocio y alcanzar a clientes en un mercado más amplio. Su problema principal es que no tienen una presencia en internet que les permita darse a conocer y gestionar sus operaciones de forma digital.

Nuestras Necesidades Clave:

  • Visibilidad y Atracción de Clientes:

    Crear una página pública para mostrar servicios y atraer nuevos clientes.

  • Gestión de Talento Eficiente:

    Implementar un sistema para que abogados interesados envíen sus hojas de vida.

  • Optimización de Operaciones Internas:

    Desarrollar una herramienta para que los abogados puedan registrar la información de sus clientes.

Requerimientos y Actores Involucrados

Requerimientos Funcionales
  • **Módulo Público:** Información detallada de servicios, noticias y perfiles de abogados.
  • **Módulo de Talento:** Plataforma para registro y envío de hojas de vida.
  • **Módulo de Gestión:** Herramienta interna para registro de clientes y seguimiento de casos.
Actores Involucrados
  • **Clientes:** Interactuar con la información pública y contactar.
  • **Profesionales Externos:** Postularse y enviar su perfil para futuras contrataciones.
  • **Abogados de la Firma:** Gestionar su cartera de clientes y el progreso de los casos.
  • **Administradores del Sistema:** Supervisar el sistema, revisar postulaciones y gestionar usuarios.

Tecnologías Recomendadas para el Desarrollo

Para garantizar usabilidad, diseño responsive y eficiencia, hemos seleccionado las siguientes herramientas:

Bootstrap

**Recomendado para este proyecto.** Es un framework completo que acelera el desarrollo, ofrece componentes prediseñados y garantiza un diseño responsive de forma automática.

Tailwind CSS (Alternativa)

Una excelente alternativa que brinda un control total sobre el diseño mediante clases de utilidad. Permite una personalización completa y un código final más ligero.
